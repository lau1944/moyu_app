# 摸鱼 🐟 APP

摸鱼专用应用 支持 MacOS Windows (Ubuntu计划中-～-) 


本项目纯属娱乐. 使用flutter开发, 兼容多系统.

## 项目构建

代码从仓库下载到本地后 

执行 Mac 应用 (release模式)

```dart
flutter run -d macos --release
```

执行 Windows 应用
```dart
flutter run -d windows --release
```
